## **实用代码及技巧大全**

**打开Windows自带的恶意软件删除工具：**
1.win键+R
2.输入`mrt`并回车

**Windows打开开发模式：**
1.创建一个文件夹
2.输入名称`GodMode.{ED7BA470-8E54-465E-825C-99712043E01C}`

**快速检查网络的畅通状态：**
1.win键+R
2.输入ping 你想要的的网站

**一键清理磁盘垃圾：**
1.win键+R
2.输入`cleanmgr`并回车

**一键修复电脑：**
1.win键+R
2.输入`dir/s`并回车

**远程控制电脑：**
1.win键+R
2.输入`mstsc`并回车

**快速查看电脑配置：**
1.win键+R
2.输入`msinfo.exe`并回车

**查看别人最近在电脑上干了什么：**
1.win键+R
2.输入`recent`即可查看
